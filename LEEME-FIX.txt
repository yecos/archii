FIX 2 — IA Real Conectada
========================================
  El asistente IA ahora usa una API real compatible con OpenAI.

  CONFIGURACIÓN EN VERCEL (Environment Variables):

  OPCIÓN A — Google Gemini (GRATIS hasta 15 peticiones/min):
    OPENAI_API_KEY = [tu clave de Google AI Studio]
    OPENAI_BASE_URL = https://generativelanguage.googleapis.com/v1beta/openai
    AI_MODEL = gemini-2.0-flash

  OPCIÓN B — OpenAI (desde $0.15/1M tokens):
    OPENAI_API_KEY = sk-[tu clave]
    OPENAI_BASE_URL = https://api.openai.com/v1
    AI_MODEL = gpt-4o-mini

  OPCIÓN C — DeepSeek (muy económico):
    OPENAI_API_KEY = [tu clave]
    OPENAI_BASE_URL = https://api.deepseek.com/v1
    AI_MODEL = deepseek-chat

  CÓMO CONFIGURAR:
    1. Ve a vercel.com → tu proyecto → Settings → Environment Variables
    2. Agrega OPENAI_API_KEY (obligatorio)
    3. Agrega OPENAI_BASE_URL y AI_MODEL (opcional, hay defaults)
    4. Guarda y re-despliega

  NOTA: Si no configuras la API key, el asistente mostrará instrucciones
  claras dentro de la app sobre cómo hacerlo.
